## Requirements

#### [REQ_0001] USB serial

Firmware should provide a basic USB serial.

#### [REQ_0002] SPI

Firmware should provide a basic SPI. SPI0 instance of RP2040 is used.

#### [REQ_0003] Echo

Firmware should echo on the SPI any data received on USB serial.
